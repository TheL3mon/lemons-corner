Particle Systems

The video above contains an assortment of my older Unreal Engine Niagara and Cascade particle systems.

Let's break down the shader used for the "magic portal" system (as seen towards the end of the video)!

The trick lies in radial panning. This is a technique that allows us to create a motion originating from a point in a circular fashion.

This is then "spiced up" by creating an outer layer of the same pattern, which we can then tint another color for an interesting effect.
The particle system that creates the white sparkles around and inside the portal was achieved by adding vortex motion to particles spawning in two rings.
