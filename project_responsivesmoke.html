Responsive Smoke

In this project I attempted to create responsive smoke inspired by [this trailer] of Counter-Strike 2.
I used the Niagara particle system and the shader graph of Unreal Engine.

The smoke is instantiated as a collection of geometrical meshes.
I chose cubes as they fill blocky environments more accurately, such as the corners of rooms or buildings.
The smoke's volume is defined by the number of blocks instantiated and their density. These create the radius of the area they attempt to fill.

In order to simulate the dynamic dispersion of smoke, I'm setting the collision handling override to automatic adjustment,
which makes sure that smoke blocks that would spawn inside shapes get moved out to a free area instead.

The smoke itself was created with the Niagara particle system.
To simulate light scattering and smoke density visually, the material of the particles has alpha blending enabled.

Furhtermore, to ensure that the billboards don't create artifacts when interacting with surfaces, I'm utilizing depth fade.
This node uses the depth buffer to calculate a threshold where the billboards would fade out seamlessly.
